# ai-course

# To remember

Add the open ai key in the .env file in the format

```
OPENAI_API_KEY='...'
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## Chat GPT clone

run

```bash
bun chat.ts
```

## Semantic Search

run

```bash
bun search.ts
```

## Documents qa

run

```bash
bun qa.ts "question"
```
